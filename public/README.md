# Public Directory

This directory contains static assets for your Next.js application.

## Audio Files

To enable sound notifications in the Recipe Timer:

1. Add a `notification.mp3` file to this directory
2. You can find free notification sounds at:
   - [Freesound.org](https://freesound.org/)
   - [Zapsplat](https://www.zapsplat.com/)
   - [SoundBible](http://soundbible.com/)

## Image Files

All recipe category images are stored in the `images/` subdirectory.

## Video Files

Cooking videos are stored in the `videos/` subdirectory.
